# MVP Scope Lock

This document is the default boundary for MVP work. A task may change it only through an explicit product decision and corresponding harness update.

## In MVP

- A 10–12 turn, single-player game targeting a 45–75 minute session.
- Predefined archetype selection.
- Three AI rival civilizational projects.
- Structured player actions.
- Deterministic simulation.
- Internal factions and institutions.
- Contradictions and legitimacy pressure.
- External relations and rival reactions.
- State-driven event cards.
- Ending states.
- A visual dashboard/situation-room concept.
- Core gameplay with no LLM dependency.
- Core gameplay with no Supabase dependency.

## Out of MVP

- A real-world map, historical countries, or direct modern geopolitical mappings.
- Multiplayer.
- Save, load, or replay.
- Authentication.
- Supabase-backed persistence.
- A fully free-form LLM game master.
- Global random shocks.
- A full technology tree.
- A detailed economy or production-chain simulator.
- Tactical war.
- Full parliament simulation.
- Mobile-specific UI.
- Production-grade analytics.

The full ideology constructor, deployment automation, and final naming system are also deferred unless a later task explicitly revises the scope.
